# Evidence Package Export Contract

## Status

Phase 14 reviewable export foundation.

## Export Posture

Evidence package export is represented as reviewable JSON metadata only.

## Manifest Fields

Each manifest records:

- export manifest ID
- evidence package ID
- evidence chain ID
- tenant ID
- workspace ID
- export status
- export type
- reviewer decision
- package integrity status
- included artifacts
- redaction status
- boundary statement
- SOC 2 mapping

## Boundary

No production export, signed bundle, regulated customer evidence package, auditor attestation, or production operating effectiveness claim is generated in this phase.
