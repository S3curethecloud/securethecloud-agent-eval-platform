# Enterprise Preview Website Positioning

## Status

Phase 18 / Doctrine-safe public positioning added.

## Purpose

This document defines doctrine-safe enterprise preview website language for the SecureTheCloud Agent Evaluation Platform.

## Canonical positioning

SecureTheCloud Agent Evaluation Platform is a governed Phase 2 AI Chaos Harness / offline evaluation support repository.

It helps evaluate autonomous agent behavior, benchmark outcomes, resilience evidence, policy candidates, RAG grounding, tool-call behavior, regression drift, and reviewer-ready evidence packages.

It is not a new product suite, not a fifth suite, not an enforcement surface, not a runtime controller, and not a live production mutation system.

## Enterprise preview mode

Enterprise preview mode means:

- frontend website positioning is available;
- lab-safe evaluation evidence is visible;
- doctrine boundaries are visible;
- reviewer-oriented evidence surfaces are visible;
- Cloudflare preview readiness language is allowed.

Enterprise preview mode does not mean:

- TRUE_MODE is active;
- production authority is granted;
- live autonomous execution is active;
- production customer data is processed;
- SOC 2 certification is claimed;
- production operating effectiveness is claimed.

## Audience

The website may speak to:

- AI governance leaders;
- CISOs;
- compliance and audit leaders;
- AI platform owners;
- agent framework teams;
- reviewers and evidence custodians.

## Public posture

The website may position the platform as an enterprise preview and evidence-readiness surface.

It must not position the platform as a customer-offerable fifth suite or enforcement authority.
